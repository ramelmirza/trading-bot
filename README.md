# trading-bot

This program sends GET requests to CoinMarketCap's API and stores the JSON data inside a csv file.
<br>
**Data:**
  - Price
  - 24H Volume
  - 24H % Change
  - Market Cap
  - Market Cap Dominance
  - Date of Last Update
<br>
